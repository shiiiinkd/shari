# handoff — Claude 間引き継ぎメモを生成する

## 目的

現在の会話を、次の Claude セッションがそのまま続きから作業できる形に圧縮する。

## 出力先

`.handoff/handoff-YYYY-MM-DD-HHMM.md`（分単位タイムスタンプ・常に新規作成）

## 含める内容

- 何を作っているか
- 決定した設計
- 現在の状態
- 次にやること

簡潔に、でも完全に。

## ターミナル出力

保存したパス1行のみ。本文は echo しない。
