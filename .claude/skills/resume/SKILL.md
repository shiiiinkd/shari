# resume — 直近の handoff から作業を再開する

## 目的

`/handoff` で保存された直近の引き継ぎメモを読み、そこに書かれた「次にやること」を即座に実行に移す。

## 手順

1. 最新の handoff ファイルパスを特定する:
   ```bash
   ls -t .handoff/handoff-*.md | head -1
   ```
2. そのファイルを Read で読み込む
3. 「現在の状態」を踏まえ、「次にやること」の先頭から作業を開始する

## 出力

着手前に1行だけ宣言する:

```
<path> を読み込みました。次のアクション: <次にやることの先頭1項目>
```

その後すぐに実装に入る。
